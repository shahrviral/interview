Dictionary Operations:

| Operation | Time Complexity | Example |
| --------- | :---------------: | ------- |
| Copy | O(n) | d.copy() |
| Get | O(1) | d.get(key) |
| Set | O(1) | d\[key\] = val|
| Delete | O(1) | del d\[key\] |
| Contains | O(1) | key in d |
| Iterate | O(n) | i for i in d.items() |
